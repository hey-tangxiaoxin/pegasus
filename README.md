# Pegasus

Web3 DApp - 多链钱包连接与 ERC20 代币转账

## 功能

- **钱包连接** - 支持 EIP-6963 标准，兼容 MetaMask、OKX Wallet 等主流钱包
- **多链支持** - 9 条主网 + 9 条测试网（Ethereum、Polygon、Arbitrum、Optimism、BSC、Base、Avalanche、zkSync、Linea）
- **账户管理** - 显示余额、ENS 域名、ERC20 代币持仓
- **代币转账** - 支持任意 ERC20 代币转账

## 技术栈

React 19 · TypeScript · Vite · Ant Design 6 · ethers.js 6

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建
pnpm build
```
